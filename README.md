# Shady Shanab — Full‑Stack Web & Mobile Developer | Portfolio

Personal portfolio for showcasing web and mobile projects, case studies, and contact. Built with Next.js (App Router), TypeScript, and SCSS with smooth animations and SEO‑ready metadata.

- Live: https://shadyshanab.tech
- Portfolio: `/portfolio-details/[id]`
- Blog: `/blog`
- Contact: `/contact`

## Tech Stack
- Next.js (App Router) + React + TypeScript
- SCSS, GSAP animations
- Image optimization via `next/image`
- SEO: per‑route `metadata`, Open Graph, Twitter Cards, JSON‑LD

## Features
- Project portfolio for Next.js, React, React Native, and WordPress work
- Dynamic portfolio details by `id`
- Blog listing and details
- Contact page with CTA
- High‑performance, responsive UI

## Getting Started
Prerequisites:
- Node.js `>= 18.18`
- npm
